Tutor Services Website

For this project, we were tasked with creating a website with tight time restrictions. We had a single day to create this website to the specifications supplied. The website has two sides.

The first side is what users are able to submit requests for tutors to the site. They supplied their name, what class they needed a tutor for, and other information and the request would be on display for potential tutors.

On the tutor’s side, the users can see any ongoing tutor requests that have been submitted. They are able to view the requests and respond with an offer to tutor. When they make an offer, an automated email with the tutor’s contact information is sent to the tutee.

The web serve has a backend database that stores all user information and tutor requests.
