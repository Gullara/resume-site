RPG Companion App

The RPG Companion App was made to store vital character information for playing RPG games. Its primary purpose is to store vital information about a user’s character for quick reference during play.

A user simply makes a new character sheet and indicates the character’s name and the type of game they are playing. Right now the app only supports one type of game, but it’s built with the intention of supporting more. From there the user inputs the required information into the sheet and saves it. It can now be accessed from the character list at any time and edited as needed.

The app also has a dice rolling function. The user can select a type of die from several types and then shakes the device. The result of the roll is displayed on the screen.
