eVAP Trainer Module

For this project, we were given a scenario where the client needed an application to train their new employees. They needed to model launching a package to particular drop points. The program needed to be able to realistically represent the package’s flight through the air. This included taking factors such as drag, weight of the package, launch angle, a thruster on the package, and more.

The left hand side of the main window contains a number of input fields that the trainee would enter to dictate the flight of the package. On the right hand side are a number of display fields for the trainee’s information. Most importantly, the center of the window contains a panel that depicts the launch area, as well as the drop points. As the package moves through the air, its trajectory is depicted in real time. The trainee is scored on their accuracy, speed, and how gentle the landing was.

There is a second function in the program that allows trainers to create new scenarios for trainees to practice with. The trainer can set environmental conditions, draw the terrain, and place the drop points. These scenarios are saved to a file which is loaded in the main application.
