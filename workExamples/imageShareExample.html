Image Sharing Website

This website is an image sharing website that allows registered users to upload images and comment on other users’ images. The home page displays the most popular images as thumbnails. There is a registration page that allows users to sign up and fill in some personal information. After they’ve registered they can upload their images. These images each have their own page that displays the image as well as its comments.

Administrators have their own page. There they can see any images users have reported as being inappropriate or otherwise against the site’s rules. On the individual image pages, administrators can choose to remove a picture if they deem it necessary.

All the images and user data is stored in a database on the server that hosts the website. In addition to creating the website itself, we also created the database.
